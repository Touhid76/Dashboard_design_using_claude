# 📊 Dashboard Design using Claude AI

 A smart and structured approach to designing **clean, insight-driven dashboards** using AI-powered prompt engineering.

This repository focuses on creating **simple, beginner-friendly, and business-ready dashboards** for analyzing student performance data.

---

## Project Goal

To design dashboards that help decision-makers quickly understand:

* 📈 What drives high performance (CGPA)
* ⚠️ What causes poor performance
* 🚨 Which students are at risk

---

## 🧠 Key Approach

* AI-powered prompt design using Claude
* Focus on **insights over complexity**
* Clean and minimal dashboard structure
* Business-oriented visualization strategy

---

## 📊 Dashboard Highlights

* 📌 Key KPIs (CGPA, Study Hours, Attendance, Risk %)
* 📉 Performance Analysis (CGPA vs Study behavior)
* 🧩 Behavioral Insights (Study vs Social Media)
* ⚠️ Risk Analysis (Probation & Suspension)
* 🧑‍💻 Resource Impact (Device usage vs CGPA)
* 🌍 Demographic Insights (Gender, Income)
